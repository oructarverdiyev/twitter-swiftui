# Twitter
Twitter clone in SwiftUI with MVVM, 
Firebase for Login/Register and other database.

- Using Firebase 
  - Login with email
  - Register with email
- Firestore
  - Save all details of register user
  - Upload profile picture
- Tweet like
- Tweet bookmark
- Post a tweet
- Fetch Profile detail
- Fetch all tweets
- Fetch likes tweets
- Fetch bookmarks
- Fetch all register users
- Logout
